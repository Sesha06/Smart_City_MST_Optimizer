## 🚀 Quick Start

1. Extract `Smart_City_MST_Optimizer.zip`
2. Navigate to extracted folder
3. Run: `npm install`
4. Run: `npm start`
5. Open `index.html` in web browser

## ✨ Key Features

- **Kruskal's Algorithm**: Sort edges by weight, build MST greedily
- **Prim's Algorithm**: Expand from starting node with min-weight edges
- **Interactive Visualization**: Watch algorithms execute step-by-step
- **Graph Statistics**: Density, hub nodes, degree distribution
- **Preset Networks**: Small (6), Metro (9), Large (12) cities
- **Real-time Analysis**: Cost calculation and optimization metrics
- **API Comparison**: Run both algorithms side-by-side

## 📊 Application Results

**Small City Network (6 nodes, 10 edges):**
- Total Cost: 65
- MST Cost: 23
- Savings: 42 (65% optimization)
- MST Edges: 5 (no redundancy)
